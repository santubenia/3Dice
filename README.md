# 3D Dice Rotation Game

Simple web-based 3D dice that spins and displays a number from 1–6 after each complete rotation.

Files added:
- index.html
- src/main.js
- src/styles.css

To run locally, open `index.html` in a browser or serve the folder with a simple HTTP server:

```bash
# from workspace root
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```

Click the `Roll` button or click the dice to spin it.
